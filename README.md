# ImageFeatGen

ImageFeatGen is a command line tool used to calculate hashes
for images. An card image fetcher is embedded to get card thumb
images from [Bestdori!](https://bestdori.com).

Add `ImageFeatGen.xcodeproj` to a Xcode Workspace
with [DoriKit](https://github.com/Greatdori/DoriKit)
to build this tool.
